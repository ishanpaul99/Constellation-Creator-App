# Constellation Creator

Constellation Creator is a fantasy-inspired celestial design application built using HTML, CSS, and JavaScript. It allows users to place stars, connect them into constellations, create unique celestial legends, and build a personal gallery of stories written across the night sky.

Combining creativity, storytelling, and visual design, the application transforms simple star patterns into mythical constellations complete with names, rarity levels, lore, and statistics.

## Features

### Interactive Night Sky

Create your own constellations by:

- Placing stars
- Moving stars
- Removing stars
- Connecting stars together

Design unique celestial formations directly on an interactive star map.

### Constellation Creation

Customize each constellation with:

- Name
- Type
- Rarity
- Mythological Story

Every creation becomes part of your personal celestial collection.

### Constellation Types

Choose from:

- Celestial Beast
- Hero
- Ancient Symbol
- Mythical Creature
- Cosmic Guardian
- Unknown Entity

### Rarity System

Assign rarity levels:

- Common
- Rare
- Epic
- Legendary
- Mythic

Rarity influences the prestige and uniqueness of each constellation.

### Myth Generator

Generate fantasy-inspired celestial legends.

Example:

> "The Silver Fox guides lost travelers beneath moonlit forests."

Other generated myths may describe ancient guardians, forgotten kingdoms, cosmic dragons, and celestial spirits.

### Name Generator

Generate constellation names such as:

- Silver Fox
- Eternal Serpent
- Astral Wolf
- Void Phoenix
- Lunar Crown
- Sapphire Dragon
- Golden Stag

### Constellation Gallery

Store and manage all created constellations.

Saved information includes:

- Name
- Type
- Rarity
- Myth
- Date Created

The gallery acts as a personal celestial archive.

### Statistics Dashboard

Track:

- Total Constellations
- Average Stars Per Constellation
- Most Common Type
- Most Common Rarity

Statistics update automatically as new constellations are created.

### Star Rating System

Each constellation receives a sky rating based on:

- Number of stars
- Complexity
- Connections

Possible results include:

- Simple Pattern
- Emerging Legend
- Celestial Masterpiece

### Favorites System

Mark special constellations as favorites and quickly access your best creations.

### Random Constellation Generator

Instantly generate:

- Random star patterns
- Random connections
- Random names
- Random myths

Perfect for inspiration and experimentation.

### Export Feature

Export completed constellations as PNG images.

Preserve the appearance of the night sky and share your creations.

### Local Storage Support

All saved constellations remain available after refreshing the page.

Stored data includes:

- Constellations
- Favorites
- Statistics
- Gallery Entries

## Technologies Used

- HTML
- CSS
- JavaScript
- Canvas API
- Local Storage

## Development

The initial codebase was generated with the assistance of GitHub Copilot as part of an AI-assisted development workflow.

Additional modifications and refinements were implemented manually, including constellation editing systems, myth generation improvements, statistics tracking, gallery management, responsiveness enhancements, export functionality, UI polishing, and user-experience improvements.

## Visual Design

Constellation Creator features:

- Animated night sky backgrounds
- Twinkling stars
- Nebula-inspired gradients
- Celestial glow effects
- Glassmorphism cards
- Smooth animations
- Fantasy-inspired styling

The application is designed to feel magical, creative, and immersive while encouraging users to tell stories through the stars.
